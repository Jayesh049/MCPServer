Agents Assemble — 20-Disease MCP Tester

A manual testing UI for the Agents Assemble Healthcare MCP server. All data is synthetic and fictional — no real PHI ever enters the system.

The platform demonstrates a full AI pipeline: an image, form, or signal input is processed by a disease-specific model, escalated through a risk-resolution stage, and concluded with actionable solution steps and a synthetic care plan.

Pipeline Architecture
Input (image | form | signal)
 → Model Inference (CNN | tabular | signal-ML)
 → Prediction (classification, confidence, risk level)
 → Resolution (risk-stratified steps)
 → Solution (actions, follow-up, patient ed.)
 → Care Plan (doctors, exercises, affirmations)
